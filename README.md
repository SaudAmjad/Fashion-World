Fashion World is a modern and responsive e-commerce frontend website designed for showcasing clothing, watches, and fashion accessories. The project focuses on creating a clean user interface and smooth shopping experience using only HTML and CSS.

The website includes essential e-commerce features such as product listings, category filters, pricing sections, ratings, navigation menus, and a newsletter subscription popup. The design emphasizes usability, responsive layout, and modern UI/UX principles.
